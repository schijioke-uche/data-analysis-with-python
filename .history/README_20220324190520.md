## Usage:   
Usage with Hypothesis data analysis for non-textual/thematic data. 
For, example, it could be use when performing dependent sample t-tests where you typically need to determine the following two hypotheses:
1. Null hypothesis (H0): that if the true mean difference is equal to zero (between the observations).
2. Alternative hypothesis(H1): that the true mean difference is not equal to zero (two-tailed).
